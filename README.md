# 背景替换相机——快速录制人像+背景混合影片

基于opencv+onnxruntime+QT的背景替换相机工具，简单好用！

本项目使用Modnet作为人像分割模型，项目地址：https://github.com/ZHKKKe/MODNet

你也可以修改代码，替换其他模型文件来实现不同的分离效果
